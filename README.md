#  Sales & Profit Dashboard (Excel)

##  Project Overview

This project presents an interactive **Sales & Profit Dashboard** built using Microsoft Excel.
The dashboard provides insights into sales performance, profitability, and regional trends for a stationery business.

---

##  Dashboard Preview

![Dashboard](dashboard.png)

---

##  Dataset & File

* `sales_dashboard.xlsx` → Contains both dataset and dashboard
* Dataset includes:

  * Product (File, Marker, Notebook, Pen, Stapler)
  * Category (Writing, Office Supplies)
  * Region (Bihar, Delhi, UP)
  * Sales & Profit data

---

##  Key Metrics

* **Total Sales:** ₹80,770
* **Total Profit:** ₹11,920
* **Profit Margin:** 15%

---

##  Features of Dashboard

* Interactive **Slicers**

  * Product filter
  * Region filter
  * Category filter

* KPI Cards

  * Total Sales
  * Total Profit
  * Profit Margin

* Visualizations

  * Sales by Category (Bar Chart)
  * Profit by Region (Pie Chart)

* Insights Panel

  * Writing category contributes majority of sales
  * Profit margin is relatively low (~15%)
  * UP region shows strong profitability
  * Opportunity to improve margins

---

##  Tools & Skills Used

---

##  Business Insights

* Writing products are the primary revenue driver
* Profitability varies significantly across regions
* Low margin suggests need for cost optimization or pricing strategy
* Regional analysis helps identify high-performing markets

---

##  How to Use

1. Open `sales_dashboard.xlsx`
2. Go to **Dashboard sheet**
3. Use slicers to filter:

   * Product
   * Region
   * Category
4. Analyze changes in KPIs and charts

---

##  Conclusion

This dashboard helps stakeholders quickly understand sales performance and identify areas for improving profitability using simple Excel-based analysis.
